üêõ TROUBLESHOOTING
--------------------------------------------------------------------------------

Issue: Page won't load
Fix: Check npm run dev is running
     Check php artisan serve is running
     Run: php artisan optimize:clear

Issue: Styles not working
Fix: Run: npm run build
     Check className (not class)
     Verify app.css loaded

Issue: Echo not connecting
Fix: Start Reverb: php artisan reverb:start
     Check .env REVERB_* variables

Issue: Module not found
Fix: Run: npm install
     Delete node_modules and reinstall
